# FTM Donation Website

A static multi-page donation website for FTM, focused on ending period poverty by empowering girls with menstrual health solutions.

## Structure

- `index.html` — Home page
- `about.html` — About Us page
- `contact.html` — Contact page
- `donate.html` — Donation page
- `get_involved.html` — Get Involved page
- `impact.html` — Impact page
- `programs.html` — Programs page
- `src/` — Images and logo assets

## Running the App

The site is served via Python's built-in HTTP server on port 5000.

**Workflow:** `Start application`  
**Command:** `python3 -m http.server 5000`

## Notes

- Styling uses Tailwind CSS via CDN (suitable for development; consider switching to the Tailwind CLI for production)
- No backend or database — fully static HTML/CSS site
